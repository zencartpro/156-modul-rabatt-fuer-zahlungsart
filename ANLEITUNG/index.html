Sinn und Zweck dieses Moduls

Mit diesem Modul kann für bestimmte Zahlungsarten ein Rabatt gegeben werden (prozentuell oder Fixbetrag)
Vor dem Einsatz dieses Moduls die neuen rechtlichen Vorschriften seit 13.01.2018 beachten:
http://www.it-recht-kanzlei.de/zahlungsarten-gebuehren-ermaessigungen-anzreizsysteme.html

Das Modul kann theoretisch auch Gebühren für Zahlungsarten verrechnen, diese sind seit Mitte Januar 2018 NICHT mehr erlaubt!

Installation

Voraussetzungen


Dieses Modul ist nur für die deutsche Zen-Cart Version 1.5.6 geeignet


Empfehlung:

Dieses Modul erst in einem Testsystem einbauen und dort konfigurieren/testen und endgültig an die eigenen Wünsche anpassen. Erst dann in einem Liveshop einsetzen! Sichern Sie unbedingt alle Dateien Ihres Shops per FTP und sichern Sie die Datenbank mit phpMyAdmin oder anderen geeigneten Tools!

WICHTIG
Vor dem Einbau dieser Änderungen:
BACKUP von Shop und Datenbank machen!
Keine Haftung, Verwendung auf eigene Gefahr!
BACKUP gemacht? Ok, dann weiterlesen...

Die Installation erfolgt in folgenden Schritten. Halten Sie diesen Ablauf ein!

1)
Im Ordner NEUEDATEIEN dieses Downloads den Ordner includes/templates/DEINTEMPLATE auf den Namen des im Shop aktiven Templates umbenennen (z.B. responsive_classic)

2)
Dann alle Ordner und Dateien im Ordner NEUEDATEIEN in der vorgegebenen Struktur ins Zen-Cart Verzeichnis hochladen. Dabei werden keine bestehenden Dateien überschrieben

3)
In der Zen-Cart Administration unter Module > Zusammenfassung das Modul "Rabatt für Zahlungsart" aktivieren.
Alle Einstellungsmöglichkeiten sind dort erklärt.

4) Konfiguration

Das Zahlungsmodul wird immer mit exakt dem Code angegeben, der unter Module > Zahlungsarten aufscheint
Rabatte immer mit - angeben
Mehrere Arten durch Komma (,) trennen

Beispiele:
Unterschiedliche prozentuelle Rabatte für mehrere Zahlungsarten:

Rabatt für PayPal Express: -3%
Rabatt für Vorkasse/Banküberweisung: -10%

Einstellung:

Rabatt oder Gebühr
-3%,-10%

Zahlungsarten für Rabatt oder Gebühr
paypalwpp,eustandardtransfer

Beispiel:
prozentueller Rabatt für eine Zahlungsart:

Rabatt für Vorkasse/Banküberweisung: -3%

Einstellung:

Rabatt
-3%

Zahlungsarten für Rabatt
eustandardtransfer

Beispiel:
Rabatt mit Fixbetrag von 3 Euro bei Zahlungsart Banküberweisung:

Einstellung:

Rabatt:
-3

Zahlungsarten für Rabatt:
eustandardtransfer

Hinweis:
Der Text "Rabatt für Zahlungsart" kann in includes/languages/german/modules/ot_total/ot_paymentmodulefee.php angepasst werden
